MDC (Movie Dekho Achhi) is a Python-based Bollywood movie recommendation application designed to help users quickly discover movies they’ll enjoy. The app provides a clean, modern user interface inspired by streaming platforms, where users can explore personalized movie suggestions based on their interests.
Built using PyQt5, the application features a dynamic and visually engaging UI with background image transitions, smooth fade animations, and responsive layouts. It also includes an email input system with validation, allowing users to submit their email for interaction or future recommendations.
The project focuses on combining user experience (UI/UX) with core Python logic, demonstrating concepts like event handling, animations, layout management, and basic backend integration using SMTP for email functionality.
